Esnaf Sipariş Yönetim API

Bu sayfada, Esnaf Sipariş Yönetim uygulamasının tüm API endpointlerini görebilirsiniz.

Swagger Dökümantasyonu İçin Tıklayın

Kimlik Doğrulama (Authentication)

  • POST /api/auth/register
    Kullanıcı kaydı (tradesman veya customer)
  • POST /api/auth/login
    Kullanıcı girişi
  • POST /api/auth/admin/login
    Admin girişi
  • GET /api/auth/verify
    Token doğrulama

Kullanıcı İşlemleri

  • GET /api/users/profile
    Kullanıcı profilini getir
  • PUT /api/users/profile
    Kullanıcı profilini güncelle
  • PUT /api/users/change-password
    Şifre değiştir
  • GET /api/users/orders
    Kullanıcı siparişlerini getir

Mağaza İşlemleri

  • GET /api/stores
    Tüm mağazaları listele (müşteriler için)
  • GET /api/stores/my
    Kullanıcının mağazalarını getir
  • GET /api/stores/{id}
    Mağaza detaylarını getir
  • POST /api/stores
    Yeni mağaza oluştur
  • PUT /api/stores/{id}
    Mağaza güncelle
  • DELETE /api/stores/{id}
    Mağaza sil
  • PATCH /api/stores/{id}/toggle-status
    Mağaza açık/kapalı durumunu değiştir

Kategori İşlemleri

  • GET /api/categories
    Tüm kategorileri listele
  • GET /api/categories/store/{storeId}
    Mağazanın kategorilerini listele
  • GET /api/categories/{id}
    Kategori detaylarını getir
  • POST /api/categories
    Yeni kategori oluştur
  • PUT /api/categories/{id}
    Kategori güncelle
  • DELETE /api/categories/{id}
    Kategori sil

Ürün İşlemleri

  • GET /api/products
    Tüm ürünleri listele (Admin için)
  • GET /api/products/store/{storeId}
    Mağazanın ürünlerini listele
  • GET /api/products/category/{categoryId}
    Kategoriye göre ürünleri listele
  • GET /api/products/search
    Ürün arama
  • GET /api/products/{id}
    Ürün detaylarını getir
  • POST /api/products
    Yeni ürün oluştur
  • PUT /api/products/{id}
    Ürün güncelle
  • DELETE /api/products/{id}
    Ürün sil
  • PATCH /api/products/{id}/toggle-status
    Ürün aktif/pasif durumunu değiştir

Sipariş İşlemleri

  • GET /api/orders
    Kullanıcının siparişlerini listele
  • GET /api/orders/store/{storeId}
    Mağazanın siparişlerini listele
  • GET /api/orders/reports/daily
    Günlük sipariş raporu
  • GET /api/orders/reports/monthly
    Aylık sipariş raporu
  • GET /api/orders/{id}
    Sipariş detaylarını getir
  • POST /api/orders
    Yeni sipariş oluştur
  • PATCH /api/orders/{id}/status
    Sipariş durumunu güncelle
  • PATCH /api/orders/{id}/cancel
    Siparişi iptal et

Admin İşlemleri

  • GET /api/admin/dashboard
    Admin dashboard verileri
  • GET /api/admin/users
    Tüm kullanıcıları listele
  • GET /api/admin/stores
    Tüm mağazaları listele
  • GET /api/admin/orders
    Tüm siparişleri listele
  • PATCH /api/admin/users/{id}/ban
    Kullanıcıyı yasakla/yasağı kaldır
  • PATCH /api/admin/stores/{id}/approve
    Mağazayı onayla/onayı kaldır
  • POST /api/admin/users
    Yeni admin kullanıcı oluştur
  • GET /api/admin/reports/overview
    Genel sistem raporu
  • GET /api/admin/reports
    Raporlar için veri getir
  • GET /api/admin/reports/export
    Rapor dışa aktarma (PDF/Excel)
